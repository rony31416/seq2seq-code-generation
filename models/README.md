# Model Files

## Download Instructions

Model files (270MB total) are stored separately due to GitHub size limits.

### Google Drive Download

**Link**: [Download All Models](https://drive.google.com/drive/folders/YOUR_FOLDER_ID)

### Required Files

Place these files in the `models/` directory:

```

models/
├── vanilla_rnn_best.pt          (90 MB)
├── lstm_best.pt                 (90 MB)
└── lstm_attention_best.pt       (90 MB)

```

### Verification

After downloading, verify files:

```bash
ls -lh models/
# Should show three .pt files
```
